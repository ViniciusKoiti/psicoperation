# psiops_contracts
Contratos da API do PsiOps — SaaS de gestão financeira para psicólogas
solo brasileiras, com foco em mensalidades.

Esta especificação é a fonte única de verdade dos contratos entre os
frontends web (landing e clinic), o app mobile (Flutter) e o backend
(Spring Boot). Os tipos consumidos pelas aplicações vêm exclusivamente
do codegen comitado em `gen/` — nunca duplicados à mão.

Domínios cobertos nesta versão: `common` (erros RFC 9457, paginação,
dinheiro, datas), `auth` (registro, login, refresh e sessão) e `lead`
(lista de espera da landing page). O domínio clínico/financeiro
(pacientes, mensalidades, cobranças) chega em tarefa futura (PSI-020).
